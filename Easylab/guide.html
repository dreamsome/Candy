Getting Started

1. Install Easylab in Your Code

First you need to add some lines in your code:

Python

import easylab.easylab as e

easylab = e.Easylab("fibonacci") 
easylab.log("n=%d, time=%f" % (n, time))     # log it

C++

#include "easylab.h"

Easylab* e = new Easylab("fibonacci");
e->log("n=%d, time=%f", n, time);      // log it

2. Fire up the Console

Then you can do everything in an interactive console. Assume you are in the directory of Easylab:

$ python scripts/console.py
=============================================
easylab 0.1.0
Type "help" for more information.

3. Run

The program fibonacci takes only one argument n, which is from 1 to 30.

easylab> autorun fibonacci 1:30

Now you can check out the results in database:

easylab> show fibonacci
        time|            n| 
    0.028133|            1| 
    0.028849|            2| 
    0.031948|            3| 
    0.031948|            4| 
    0.037193|            5| 
    0.035048|            6|
...

4. Visualize!

Say if you want see how the argument n effects the time, use plot command.

easylab> plot N, time from fibonacci

Then you get this on the screen:
