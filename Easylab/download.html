Source Code

Easylab is on Github. You may check out the current version by running:

git clone git://github.com/onesuper/Easylab.git

Then you can build from source using:

$ python setup.py install

Dependencies

Easylab uses SQLite3 and matplotlib to manage and visualize data. So make sure you have already installed both of them on your computer.

On Ubuntu/Debian, you could install them by running:

$ sudo apt-get install matpotlib, sqlite3
